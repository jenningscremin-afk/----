向量的表示

  • 向量 = 大小 + 方向
  • 表示方式：有向线段、坐标三元组
  • 模长：|a| = √(a_x² + a_y² + a_z²)

点积的含义

  • 点积公式：a·b = |a||b|cosθ = a_x b_x + a_y b_y + a_z b_z
  • 几何意义：投影与夹角
  • 判断垂直：a·b = 0 → 正交

叉积的含义

  • 叉积：a × b = |a||b|sinθ · n（n 为垂直于平面的单位向量）
  • 模长：等于由 a、b 构成的平行四边形面积
  • 方向：右手法则

  • 求面积：|a × b|
