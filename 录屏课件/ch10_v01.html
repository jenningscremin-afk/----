  • 标题：行列式
  • 提问：行列式到底“算”了什么？

动画执行提示

  1. 2×2 和 3×3 数阵漂浮出现。

基本定义

  • 2×2 行列式：ad − bc
  • 3×3 行列式：Sarrus 法示意
  • 性质：交换行变号、倍乘某行同倍数、两行相同则为 0

动画执行提示

  1. Sarrus 箭头动画展示正负项组合。
  2. 性质以示例矩阵演示。

拉普拉斯展开

  • 任意一行或一列展开
  • 示例：沿第一行展开得到 a11C11 + a12C12 + a13C13

动画执行提示

  1. 展开树逐层展开，删除的行列被灰化。

几何意义

  • 2×2 行列式的绝对值 = 线性变换后平行四边形面积
  • 3×3 行列式的绝对值 = 平行六面体体积
  • 行列式 = 缩放因子

动画执行提示

  1. 单位正方形经矩阵变换成平行四边形，面积标注为 |det A|。
  2. 立方体变为平行六面体，体积标注。

判别作用

  • det A = 0 → 向量线性相关 → 不可逆
  • det A ≠ 0 → 可逆 → 方程组有唯一解
