公式与口诀

  • 公式：$\int u dv = u v - \int v du$
  • 选择口诀：“反对幂三指”
  • 反函数（ln、arctan 等）优先做 u
  • 对数、指数、幂函数按顺序选
  • 三角函数最后考虑

基础例题

  • 例：$\int x e^x dx$
  • 选择：u=x、dv=e^x dx
  • 计算：uv−∫v du = x e^x − ∫ e^x dx = x e^x − e^x + C

反三角例题

  • 例：$\int \ln x dx$
  • 选：u=ln x，dv=dx
  • 结果：x ln x − x + C

练习

  • 练习：$\int x \sin x dx$
  • 选：u=x，dv=sin x dx
  • 答案：−x cos x + sin x + C
