导数是什么？

瞬时变化率

导数就是函数在某一点的瞬时变化率。

  • 速度是位置的变化率
  • 加速度是速度的变化率
  • 成本变化率告诉我们增产的代价
$f'(x) = \lim\limits_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}$

通俗地说：导数告诉我们函数在某一点变化的快慢和方向。

导数的几何意义

导数 $f'(x_0)$ 是曲线 $y=f(x)$ 在点 $(x_0, f(x_0))$ 处的切线斜率。

切线方程：$y - f(x_0) = f'(x_0)(x - x_0)$

斜率与函数行为

  • $f'(x) > 0$：切线向上倾斜，函数上升
  • $f'(x) < 0$：切线向下倾斜，函数下降
  • $f'(x) = 0$：切线水平，可能是极值点

用导数判断单调性

导数的符号决定了函数的增减性：

若 $f'(x) > 0$，则 $f(x)$ 在该区间单调递增 ↗

若 $f'(x) < 0$，则 $f(x)$ 在该区间单调递减 ↘

判断步骤

  1. 求导数 $f'(x)$
  2. 解不等式 $f'(x) > 0$ 或 $f'(x) < 0$
  3. 确定单调区间

单调性判断实例

例：判断 $f(x) = x^3 - 3x$ 的单调性

解题步骤

1. 求导：$f'(x) = 3x^2 - 3 = 3(x^2 - 1)$

2. 令 $f'(x) = 0$：$x = \pm 1$

3. 判断符号：

  • $x < -1$ 时，$f'(x) > 0$，递增
  • $-1 < x < 1$ 时，$f'(x) < 0$，递减
  • $x > 1$ 时，$f'(x) > 0$，递增

极值

什么是极值？

极值是函数在局部范围内的最大值或最小值。

  • 极大值：比周围所有点都大
  • 极小值：比周围所有点都小
极值点处：$f'(x) = 0$ 或 $f'(x)$ 不存在

注意：$f'(x) = 0$ 的点不一定是极值点！

极值判断方法

第一判别法（导数符号法）

在 $x = x_0$ 处，如果：

  • $f'(x)$ 由正变负 → 极大值
  • $f'(x)$ 由负变正 → 极小值
  • $f'(x)$ 不变号 → 无极值

步骤

  1. 求 $f'(x) = 0$ 的点
  2. 判断这些点两侧导数符号
  3. 确定极值点和极值

最大值与最小值

在闭区间 $[a, b]$ 上，连续函数必有最大值和最小值。

最值可能出现在：

  • 区间内的极值点
  • 区间端点

求最值步骤

  1. 求出所有极值点
  2. 计算极值点和端点的函数值
  3. 比较大小，确定最值

应用：成本最小化

问题

某工厂生产成本函数为：

$C(x) = x^2 - 10x + 100$

其中 $x$ 是产量（百件），$C$ 是成本（万元）。

求：最小成本是多少？对应产量是多少？

解答

1. 求导：$C'(x) = 2x - 10$

2. 令 $C'(x) = 0$：$x = 5$

3. 最小成本：$C(5) = 75$ 万元

应用：利润最大化

问题

某产品的利润函数为：

$P(x) = -x^2 + 12x - 20$

其中 $x$ 是销售量（千件），$P$ 是利润（万元）。

求：最大利润是多少？

解答

1. 求导：$P'(x) = -2x + 12$

2. 令 $P'(x) = 0$：$x = 6$

3. 最大利润：$P(6) = 16$ 万元

应用：容器设计

问题

用一张边长为 20cm 的正方形铁皮，四角各剪去相同的小正方形，做成无盖容器。

问：剪去多大的正方形，容器容积最大？

解答

设剪去边长为 $x$ cm，则：

容积 $V(x) = x(20-2x)^2$

求导后得：$x = \frac{10}{3}$ cm

最大容积：$V(\frac{10}{3}) \approx 592.6$ cm³

曲线的凹凸性

二阶导数 $f''(x)$ 决定曲线的弯曲方向：

$f''(x) > 0$：曲线凹向上 ∪

$f''(x) < 0$：曲线凹向下 ∩

拐点

凹凸性改变的点叫拐点。

在拐点处：$f''(x) = 0$ 或不存在

函数图像描绘步骤

  1. 定义域：确定函数存在的范围
  2. 关键点：
    • 与坐标轴的交点
    • $f'(x) = 0$ 的点（可能极值）
    • $f''(x) = 0$ 的点（可能拐点）
  3. 单调性：根据 $f'(x)$ 的符号
  4. 凹凸性：根据 $f''(x)$ 的符号
  5. 渐近线：水平、垂直渐近线

总结

导数应用要点

  • 导数 = 变化率 = 切线斜率
  • $f'(x) > 0$ → 递增；$f'(x) < 0$ → 递减
  • 极值点：$f'(x) = 0$ 且符号改变
  • 最值 = 比较极值和端点值
  • 实际问题：建模 → 求导 → 求极值

记住：导数是理解函数变化的关键工具！
